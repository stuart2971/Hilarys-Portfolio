The Lazy Beaver
Company

The Lazy Beaver Company is a auto pot watering system.  In the past, this company had an Austrailian website.  It was only until recently when they decided to make a Canadian one.  

I was asked to create a new logo, as well as fix up some images they wanted to use in the website.

I tried to keep the logo very simplistic to match the website.  After working with the web developer on the logo, we decided on a colour palette together, so while we can both work  to make all parts of the website match.

Since the company is called the Lazy Beaver, I tried to create an image of a sleeping beaver, with the hammock being ‘L’ shaped and the beaver’s body being ‘B’ shaped.

The Lazy Beaver Company sells hydroponic pots that do not require any pumps or electricity.  These can be very complicated to set up, so I was instructed to make a clear, straight forward colour coded instruction sheet to put up on the website.
‍
The web developer for this website and I were given an image of a map, though, we both agreed to make a few adjustments in order for it to match the modern and sleek website design.

We decided to change the rectangles to ovals and change the cannabis leaves to the LBC logo.  Also, get rid of the lines pointing to each province/territory, as we thought that the titles were enough.
